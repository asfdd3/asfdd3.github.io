socket系统调用

发表于 2025-05-13 更新于 2025-05-26 分类于 网络协议栈源码学习 阅读次数： Valine：

简介

socket 是网络编程中最基本的系统调用之一，用于创建一个网络通信的“端点”（即套接字，socket）。

一、函数原型

参数 说明
domain 协议族，比如：AF_INET 表示 IPv4
type 套接字类型，比如：SOCK_STREAM 表示 TCP
protocol 指定使用的协议，通常填 0 让系统自动选择

二、函数调用流程

用户态程序调用 socket()
↓
glibc 中封装的socket接口
↓
内部通过 syscall 指令
↓
进入内核，执行 sys_socket()
